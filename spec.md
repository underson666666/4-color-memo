## コンセプト

4色ボールペンを使った手書きメモの色分けに特化したテキストエディタ。


## 現在の仕様

基本機能はシンプルなメモ帳だが、色付け機能と最低限の装飾機能を持つ。

- Undo、Redo機能あり
- ファイル保存機能あり
- タブ機能あり
- 文字の色付けはショートカットに対応
- 太字装飾に対応
- スクリーンショットの貼り付けに対応
- 文字コードはUTF-8固定
- 改行コードは切り替え可能
  - デフォルトは `CRLF`
- ファイル保存はローカルのみ


## 編集仕様

色付け方法は2種類。

- ショートカットキー押下後、入力する文字の色を変える
- 選択した文字の色を変える

色付けショートカットは4種類。

- `Ctrl + D`: 黒
- `Ctrl + R`: 赤
- `Ctrl + B`: 青
- `Ctrl + G`: 緑

装飾ショートカットは1種類。

- `Ctrl + Shift + B`: 太字
- `Ctrl + V`: 画像貼り付け
- `Tab`: インデント
- `Shift + Tab`: アウトデント

タブ操作ショートカット:

- `Ctrl + Tab`: 次のタブへ切り替え
- `Ctrl + Shift + Tab`: 前のタブへ切り替え
- `Ctrl + N`: 新規タブ
- `Ctrl + W`: タブを閉じる

ファイル操作ショートカット:

- `Ctrl + O`: 開く
- `Ctrl + S`: 保存
- `Ctrl + Shift + S`: 名前を付けて保存

履歴ショートカット:

- `Ctrl + Z`: Undo
- `Ctrl + Y`: Redo


## ファイル仕様

独自拡張子は `4ct` (`4 color text`)。

ファイルに保存する内容は、文字列に HTML 風のタグを付けたテキストとする。他のエディタで開いた場合も、色や装飾の箇所はタグ付き文字列として確認できる。

使用する主なタグ:

- `<red>...</red>`
- `<blue>...</blue>`
- `<green>...</green>`
- `<bold>...</bold>`
- `<image>...</image>`

例:

```txt
通常の文字<red>赤い文字</red>
<image>assets/meeting/img-20260420-001.png</image>
<blue><bold>青の太字</bold></blue>
```

画像ファイル本体は `.4ct` ファイルと同じ階層の `assets/メモ名/` フォルダへ保存し、本文からは相対パスで参照する。


## タブと保存動作

- ファイルを保存せずにタブを閉じる場合は、保存確認ダイアログを表示する
- 「はい」を選ぶと保存ダイアログを表示する
- 「いいえ」を選ぶと保存せずにタブを閉じる
- ウィンドウ右上の閉じる操作でも、未保存タブがある場合は保存確認を行う
- 可能な限り OS 標準のダイアログを使う


## セッション仕様

- 開いていたタブ構成は次回起動時に復元する
- セッション復元用の情報は JSON 形式で保存する
- 保存ファイル名は `session.json`
- Windows では通常 `%APPDATA%\\com.codexlab.fourcolortext\\session.json` に保存される
- Undo / Redo の履歴はメモリ上のみで保持し、アプリ終了時に破棄する


## 技術要件

現在の実装は以下の構成を採用している。

- デスクトップアプリ: `Tauri v2`
- フロントエンド: `Vite + TypeScript`
- バックエンド処理: `Rust`
- Windows 向け配布形式: `NSIS setup.exe`

Ubuntu から Windows 向けインストーラを生成する場合は、`cargo-xwin` と `x86_64-pc-windows-msvc` ターゲットを使う。


## 補足

- Windows ビルドは未署名
- 配布時には SmartScreen 警告が出る可能性がある
